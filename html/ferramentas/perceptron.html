Treinamento de perceptron online usando javascript

Esta ferramenta foi criada a fim de estudos, para se realizar o treinamento de uma rede neural simples (perceptron, que usa apenas um neurônio artificial). Para isso, foi usada a linguagem Javascript em todo o algoritmo de treino. Para se usar a ferramenta, basta adicionar um arquivo com as entradas que serão usadas no aprendizado do neurônio, um arquivo de teste com saídas que são conhecidas, a fim de se verificar qual é a saída do perceptron.
Obs: a leitura só será feita em arquivos do tipo .txt, sendo que a entrada deve conter N+1 elementos por coluna, onde N é a quantidade de entradas definida, e o último elemento é a saída esperada (que pode valer 1 ou -1). Os elementos devem ser separados por um espaço simples.
Caso algum erro na execução do código seja encontrada, fique à vontade para entrar em contato avisando o problema.
